# PocketBook Digibib reader

for (mostly german) [volumes](https://www.versand-as.de/Digibib-Gesamtverzeichnis.html) from directmedia, currently
supporting

- [x] text
- [ ] images
- [ ] links
- [ ] toc

## Setup

copy [latest build](https://github.com/zigfield-jr/pb-digibib/releases/download/latest/pb-digibib.zip) and volumes to
B288 or B300 device via USB

```
.
├── applications
│   └── digibib.app
└── volumes
    ├── DB001
    │   ├── data
    │   └── images
    └── DB002
        ├── data
        └── images
```
